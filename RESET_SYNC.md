# 清除禅道同步时间戳 - 立即重试指南

## 问题说明
如果同步失败，时间戳不会被更新，你可以立即重试。但如果你需要强制重新同步，可以手动清除时间戳。

## 方法1：在浏览器控制台执行（推荐）

打开新标签页，按F12打开控制台，粘贴以下代码：

```javascript
chrome.storage.local.remove('zentao_last_sync_time', () => {
  console.log('✓ 同步时间戳已清除，下次加载将立即同步');
});
```

## 方法2：使用扩展重新加载

1. 打开 `chrome://extensions/`
2. 点击工作助手的"重新加载"图标
3. 打开新标签页
4. 将自动触发同步

## 验证是否清除成功

在控制台执行：
```javascript
chrome.storage.local.get(['zentao_last_sync_time'], (result) => {
  console.log('上次同步时间:', result.zentao_last_sync_time || '从未同步');
});
```

应该显示：`上次同步: 从未同步`

## 代码逻辑说明

### ✅ 成功时
```
同步成功 → 更新时间戳 → 24小时内不再自动同步
```

### ❌ 失败时
```
同步失败 → 不更新时间戳 → 立即重试或下次加载时自动重试
```
