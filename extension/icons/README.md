# 插件图标说明

## 快速生成图标

### 方式一：在线生成（推荐）

访问以下网站，输入一个 emoji 或文字即可生成图标：

1. **favicon.io**
   - 访问: https://favicon.io/
   - 输入文字或emoji（如 📝 或 "工作"）
   - 下载生成的图标

2. **Canva**
   - 访问: https://www.canva.com/
   - 搜索 "App Icon"
   - 设计并导出 PNG 格式

3. **免费图标网站**
   - https://www.iconfinder.com/
   - https://www.flaticon.com/
   - 搜索 "task", "checklist", "todo" 等关键词

### 方式二：使用系统自带工具（macOS）

1. 打开 "预览" 应用
2. 打开任意图片或选择 emoji
3. 文件 → 导出 → 选择 PNG 格式
4. 调整尺寸为 16x16, 48x48, 128x128

### 临时方案

在开发阶段，你可以使用任何图片作为临时图标：
- 下载任意方形图片
- 重命名为 icon16.png, icon48.png, icon128.png
- 放入 extension/icons/ 目录

### 推荐图标风格

- 简洁的 **✓** 对勾
- 📝 记事本
- 📋 剪贴板
- 🎯 目标
- 任何简洁的任务管理相关图标
